# Mobile Detailing Application

## Overview
This project was developed to streamline KleenPride’s booking process and improve customer and business management. It provides a dual-interface system for both clients and administrators.

## Features

### Client Side
- Browse and view available cleaning services  
- Enter and manage personal details  
- Book services through an intuitive interface  

### Admin / Owner Side
- View and manage customer bookings  
- Calculate trip expenses (to and from client locations)  
- Track income and overall business expenses  
- Manage driver availability and profiles  
- Secure admin login system  

## Purpose
The system was designed to improve operational efficiency by digitising booking, tracking, and cost management processes for a small business.

## Features
- Firebase Authentication & Secure login
- Google API & Google Maps
- Payment API 

## Technologies Used
Kotlin / Java
